Float Demo

1. Basic float left and right

This box is floated left.
This box is floated right.

This is normal text inside the same container. Notice how it wraps around the floated boxes. The floats are taken out of normal flow, so inline content flows around them.

The line above uses clear: both; to push the next content down below the floats.

2. Parent height without clear (no fix)

This parent has a floated child but no clearfix or overflow fix. The border appears to collapse because the parent does not "see" the height of the floated child.

I am floated inside a parent with no clear or overflow fix.

Notice how the bottom of the parent border is very close to the top: it did not expand to include the floated child.

3. Parent height with overflow fix

This parent uses overflow: auto; so it will expand to contain the floated child.

I am floated inside a parent with overflow: auto;, so the parent wraps around me.

4. Using clear on following content

In this example we float a small box to the left. The first paragraph wraps around it, but the second paragraph uses clear: left; so it starts below the float instead of beside it.

Small floated box

This paragraph flows around the floated box. Its text appears to the right of the small box and wraps underneath it as needed.

This paragraph has clear: left;, so it is pushed below the left-floated box. This is another way to use clear without adding an extra "clearfix" div.
